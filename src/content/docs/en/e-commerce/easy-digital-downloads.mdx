[Easy Digital Downloads](https://easydigitaldownloads.com/?ref=5375) is a popular WordPress plugin for selling digital products.

The built-in integration with Easy Digital Downloads allows you to sell the template libraries you create with Bricks.

## Selling Template Libraries with Easy Digital Downloads

### Step 1: Enable the Integration

In the WordPress admin, go to **Bricks → Yabe Ukiyo**. Switch to the **Setting** tab and enable the **Easy Digital Downloads** integration under the **Easy Digital Downloads** section.

<a data-fslightbox href="/tutorial/e-commerce/easy-digital-downloads/1.png">
![](/tutorial/e-commerce/easy-digital-downloads/1.png)
</a>

Then click the **Save Changes** button.

### Step 2: Create a Downloads

In the WordPress admin, go to **Downloads → Downloads**. 

<a data-fslightbox href="/tutorial/e-commerce/easy-digital-downloads/2.png">
![](/tutorial/e-commerce/easy-digital-downloads/2.png)
</a>

### Step 3: Configure the Downloads

Enter a title for the downloads and a description. 

<a data-fslightbox href="/tutorial/e-commerce/easy-digital-downloads/3.png">
![](/tutorial/e-commerce/easy-digital-downloads/3.png)
</a>

Then, under the **Yabe Ukiyo** section:

1. Tick the **Generate?** checkbox.

2. (Optional) tick the **Renewal?** checkbox, if you want to renew the license when a subscription renewal is purchased.

3. (Optional) fill in the **Max activations** field with the maximum number of sites allowed to access the template library using the generated license. Leave it blank to allow unlimited activations.

4. (Optional) fill the **Duration** field with the number of days the license will be valid for. To allow the license to be used indefinitely, leave it blank.

Then click the **Publish** button.
