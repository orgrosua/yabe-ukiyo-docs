[WooCommerce](https://woocommerce.com/) is the most popular e-commerce platform for WordPress.

The built-in integration with WooCommerce allows you to sell the template libraries you create with Bricks.

## Selling Template Libraries with WooCommerce

### Step 1: Enable the Integration

In the WordPress admin, go to **Bricks → Yabe Ukiyo**. Switch to the **Setting** tab and enable the **WooCommerce** integration under the **WooCommerce** section.

<a data-fslightbox href="/tutorial/e-commerce/woocommerce/1.png">
![](/tutorial/e-commerce/woocommerce/1.png)
</a>

Then click the **Save Changes** button.

### Step 2: Create a Product

In the WordPress admin, go to **Products → Add New**. 

<a data-fslightbox href="/tutorial/e-commerce/woocommerce/2.png">
![](/tutorial/e-commerce/woocommerce/2.png)
</a>

### Step 3: Configure the Product

Enter a title for the product and a description. 

<a data-fslightbox href="/tutorial/e-commerce/woocommerce/3.png">
![](/tutorial/e-commerce/woocommerce/3.png)
</a>

Then, under the **Yabe Ukiyo** section:

1. Tick the **Generate?** checkbox.

2. (Optional) tick the **Renewal?** checkbox, if you want to renew the license when a subscription renewal is purchased.

3. (Optional) fill in the **Max activations** field with the maximum number of sites allowed to access the template library using the generated license. Leave it blank to allow unlimited activations.

4. (Optional) fill the **Duration** field with the number of days the license will be valid for. To allow the license to be used indefinitely, leave it blank.

Then click the **Publish** button.
