Website design is the heart of all websites. The more attractive yet easy-to-use your WordPress website is, the more your visitors will love it. Creating a WordPress website with a beautiful design and a great user experience requires a lot of time and effort. You can save a lot of time and effort with a foundation and a good starting point.

[Bricks Builder](https://bricksbuilder.io/) has a great feature called [**Template Library**](https://academy.bricksbuilder.io/article/template-library/) which allows you to create a set of reusable design elements that can be distributed and used on another website.

<h2>What is Yabe Ukiyo?</h2>

Yabe Ukiyo is a **Remote Templates Manager** WordPress plugin made for Bricks designers and developers to reuse their template libraries in other projects. Its e-commerce integration allows you to sell your template libraries with easy.

Ukiyo is a Japanese word `浮世` (Ukiyo) that means "world".

## Key Features

The Yabe Ukiyo was carefully crafted with specific features in mind:

- <i class="fa-solid fa-key"></i> **License system** :
The complete license key generation, activation, and checking system.

- <i class="fa-solid fa-key-skeleton-left-right"></i> **Remotes manager** :
Connect to multiple remote templates sources.

- <i class="fa-solid fa-layer-plus"></i> **Truely a Bricks add-on** :
Yabe Ukiyo built only a thin layer on top of Bricks Builder. Fully compatible with Bricks' built-in template library system and make it a future-proof add-on.

- <i class="fa-solid fa-cart-shopping"></i> **E-commerce integration** :
Sell your template libraries with ease. Yabe Ukiyo is integrated with the most popular e-commerce plugins.

- <i class="fa-solid fa-feather"></i> **Lightweight and** <i class="fa-solid fa-rabbit-running"></i> **blazingly fast** :
The license system is built on Bricks' built-in system, WordPress REST API, and a modern JavaScript framework for an instant, responsive user experience. It has a small footprint and won't slow down your site.

- <i class="fa-solid fa-face-smile-hearts"></i> **Easy to use** :
Connect and manage the remote templates with a simple, intuitive interface.
